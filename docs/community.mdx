# Community Projects

The GitPOAP community is decentralized and growing. Here are some of the projects built on top of GitPOAP, by the GitPOAP community.

## GitPOAP Badge Generator For Github Profiles
This project facilitates the process of adding your GitPOAPs to your Github profile README or in any HTML/Markdown file in a simple GET request.
- [Project Github repository - try it out!](https://github.com/charmful0x/gitpoap-gh-badge)
